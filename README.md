# AuroraPay - IKT Projektmunka

Ez a weboldal a **DUE Bánki Donát Technikum 12.C osztályának** IKT projektmunka óra keretein belül jött létre. A projekt célja a pénzügyi tudatosság készségének fejlesztésére alkalmas weboldal elkészítése volt.

##Felhasznált technológiák
A projekt kizárólag frontend technológiákat használ:
* **HTML5** - Az oldal szerkezete
* **CSS3** - Stílusok és reszponzív design (mobilos nézethez)
* **JavaScript** - Az oldalon található kisebb interakciók
